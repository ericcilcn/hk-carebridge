# 港医桥/HK CareBridge · Astro + Directus 内容审阅站

这是根据「安静临床编辑感」视觉方向制作的可运行网站，定位为香港医疗信息与就医准备。前端在没有 CMS 时使用本地完整内容；配置 Directus 后，仅合并已发布记录的标题、摘要与分类，本地图片、正文、来源和更新时间保持完整。

## 页面

- 首页：一个品牌首屏，以及药物资讯、第二诊疗意见、癌症知识、微信咨询四个模块
- `/drugs/`：帕博利珠单抗、奥希替尼、安道生/环磷酰胺完整资料
- `/cancers/`：肺癌、乳腺癌、结直肠癌完整科普
- `/second-opinion/`：第二诊疗意见所需的文字概况、资料准备和医生沟通问题
- `/contact/`：四步咨询流程、微信待接入说明、联系前准备、整理内容与隐私提醒

## 内容结构

每篇文章支持摘要、头图、第二张场景图、正文分节、问题清单、权威来源、更新时间和阅读时间。药物图片来自官方或监管资料并本地托管；癌症科普图片为统一 3:2 比例的原创生成图。

## Directus 内容结构

内容模型位于 `directus/collections.json`，本地 Directus 配置位于 `directus/docker-compose.yml`。`scripts/seed-directus.mjs` 可创建核心集合及文章字段。正式环境请更换管理员账号、密码、SECRET，并配置角色权限、隐私文本和数据保存策略。

## 常用命令
